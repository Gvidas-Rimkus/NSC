This file contains the results the different mandelbrot implementations.
For this measure_performance.py has been implemented

Setup for all experiments:
    resolution: (1024, 2048)
    max_iter: 100

Reported are the 95% confidence intervals for each experiment.
The speedups are always compared to Version 1 of same system and resolution.

================PERFORMANCES==================

    GPU: None 
    RAM: LPDDR5 8GB
